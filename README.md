# Logistic Enterprise Network

## Overview

This project was developed as part of the Computer Networks course. The objective was to design and simulate an enterprise network infrastructure for a logistics organization.

---

## Features

- Enterprise network topology design
- IP address allocation
- Router and switch configuration
- End-to-end communication testing
- Network simulation using Cisco Packet Tracer

---

## Technologies Used

- Cisco Packet Tracer
- Computer Networking Concepts

---

## Concepts Implemented

- Routing
- Switching
- IP Addressing
- Network Topology Design
- Connectivity Testing

---

## Files Included

- Packet Tracer (.pkt) file
- Network topology screenshots
- Project report
